Guide de création

Tout ce qu'il faut savoir pour bien démarrer

1

Créer un cours

  • Clique sur "Créer un cours" et donne-lui un titre clair.
  • Utilise l'éditeur WYSIWYG pour structurer ton contenu (titres, paragraphes, listes).
  • Importe tes PDF, images ou vidéos en glisser-déposer.
  • Ajoute des tags (ex: Mathématiques, 3ème, Géométrie) pour l'organisation.
2

Créer un QCM / Exercice

  • Choisis le type de question : QCM, Vrai/Faux ou question ouverte.
  • Définis la bonne réponse et les points pour chaque question.
  • Active le chronomètre pour limiter le temps de réponse.
  • Consulte les statistiques après correction pour identifier les difficultés.
3

Créer un jeu

  • Sélectionne un template (Kahoot-style, quiz rapide, défis par équipe).
  • Ajoute des images, icônes et couleurs pour rendre le jeu attractif.
  • Configure le classement en temps réel et les badges à gagner.
  • Lance le jeu en mode live avec tes élèves ou en mode solo.

💡 Conseil pro

Commence toujours par structurer ton contenu avant de l'enrichir. Un bon plan = un cours réussi !
